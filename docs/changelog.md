# Changelog

PyFlexo does not currently ship a detailed release history in the repository.

## Current release line

The code in this workspace corresponds to PyFlexo 0.2.3.

The implemented public surface includes:

- WSGI application support through `PyFlexoApp`
- Function-based and class-based routing
- Template rendering with Jinja2
- Static file serving with WhiteNoise
- Global exception handlers
- Middleware wrappers
- A lightweight SQLite ORM

> **Note**
> When release notes are added to the project, list one version per heading and keep each entry limited to observable user-facing changes.
